**Key**는 영어로 열쇠를 뜻한다

* Key (기업) : 일본의 게임 회사 비주얼 아츠의 팀 브랜드
* Key (가수) : 대한민국의 가수. 샤이니의 멤버
* Key : 컴퓨터 자판의 글쇠
* Key : 일본의 애니메이션 《Key, the Metal Idol》의 주인공
* Key : 암호에서 암호화와 복호화에 사용되는 특정한 정보를 의미
* KeY - 소프트웨어 확인 도구
* .key - 키노트에 쓰이는 파일 확장자
*Key: 피아노 건반
